# [Sqwar.io](http://www.sqwar.io/)
Sqwar.io is a Realtime Web Application that lets Users Create their own Squares.

## Features
- Create Squares
- Move Squares Around
- Resize Squares to your Liking
- Make Squares your Favorite Color
- Add Text to Squares
- Add Images to Squares
- Make your own Room

## Technologies Used
1. Node.js
2. Express.js
3. MongoDb / Mongoose
4. Jquery / JqueryUI
5. Socket.io
